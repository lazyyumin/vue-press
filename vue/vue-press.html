# vue-press

# base

   默认为/ 如果想部署到github或者gitee上，需要把base修改为仓库名

// config.js
module.exports = {
  base: '/'   //  default
  //base: '/<REPO NAME>'
}

# sidebar侧边栏

# sidebarDepth

  vue-press的sidebar生成的目录默认只能自动捕获二级标题。 可以设置sidebarDepth来改变捕获深度,设置范围是0-2，0表示禁止,1自动捕获二级标题,2自动捕获二级和三级标题

//  config.js
themeConfig: {
  sidebarDepth: 2
}

# 为不同页面单独配置导航栏

可以在sidebar里设置路径

//  themeConfig
sidebar: {
  //  写法1
  '/path1/': [
    'first' //  /path1/first.md
    'second'  //  /path1/second.md
  ],
  //  写法2
  '/path2/': [
    title: 'title',
    collapsable: false,
    children: [
      ['', '导航'],
      ['react-1', '第一节']
    ]
  ]
  //  /必须写在最后
  '/': [
    ''  // /README.md
  ]
}

# sidebar重命名

可以把链接的文字写成一个数组的形式来进行重命名

sidebar: [
  ['/pages-a', 'rename']
]
